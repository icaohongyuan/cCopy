cCopy
============================

[cShare](https://plugin.caohongyuan.cn/copy) - 超轻量级复制文本插件，简洁易用、开源免费。

### 使用方法：

```html
<!-- 引入插件js -->
<script src="https://dev.caohongyuan.cn/ccopy/ccopy.js"></script>

<!-- 加载logo -->
<div id="ccopy">你要复制的内容</div>
```
更多示例请点击：https://plugin.caohongyuan.cn/copy
